Sistema operativo:
Es el software principal que administra los recursos del computador y permite ejecutar programas.

Node.js:
Es un entorno de ejecución que permite ejecutar código JavaScript fuera del navegador.

Aplicación:
Es el programa desarrollado por el usuario, en este caso el servidor creado con Express.

Puerto:
Es un punto de comunicación que permite al servidor recibir solicitudes. En esta actividad se utiliza el puerto 3000.

Logs:
Son los mensajes que aparecen en la terminal para informar el estado y funcionamiento del servidor.
